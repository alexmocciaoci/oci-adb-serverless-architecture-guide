OCI Autonomous Database Serverless – Architecture and Security Guide

This repository contains technical documentation and configuration guides for Oracle Autonomous Database Serverless (ADB-S) in Oracle Cloud Infrastructure.

The material is intended for database engineers, cloud architects and security architects designing enterprise-grade OCI environments.

Topics covered include:

• ADB-S architecture and service components
• Network architecture with Private Endpoint
• Zero Trust Packet Routing (ZPR) implementation
• IAM Database Token authentication model
• SQL Developer, SQL*Plus and Instant Client configuration
• Oracle Data Safe integration and auditing architecture
• Control Plane vs Data Plane logging model
• Monitoring and event-driven operational alerts
• Backup architecture and recovery model
• Enterprise onboarding process for database services

The documentation reflects practical implementation scenarios and architectural patterns used in real OCI environments.

All customer-specific information has been removed and replaced with generic placeholders.
